# AI学习网站操作说明

## 1. 启动网站

打开 PowerShell，进入项目目录：

```powershell
cd C:\Users\24424\Desktop\ai学习网站
```

启动开发服务器：

```powershell
npm.cmd run dev -- --hostname 127.0.0.1 --port 3000
```

看到类似下面的提示，说明启动成功：

```text
Local: http://127.0.0.1:3000
Ready
```

然后在浏览器打开：

```text
http://127.0.0.1:3000
```

如果想停止网站，在 PowerShell 窗口里按：

```text
Ctrl + C
```

## 2. 管理员登录

网站可以公开浏览，但上传、编辑、删除、备份等管理操作需要管理员密码。

进入：

```text
http://127.0.0.1:3000/settings
```

输入项目根目录 `.env` 文件里的 `ADMIN_PASSWORD`。

如果要修改密码，打开 `.env` 文件，修改：

```env
ADMIN_PASSWORD="你的新密码"
```

修改后需要重新启动网站。

## 3. 首页：课程仪表盘

首页地址：

```text
http://127.0.0.1:3000
```

首页可以查看：

- 课程数量
- 课时完成进度
- 总学习时长
- 课程列表
- 待完成任务
- 错题提醒

也可以在首页直接新建课程。

## 4. 创建课程、章节和课时

### 创建课程

在首页右侧的“新建课程”区域填写：

- 课程标题
- 课程描述

点击“新建课程”。

### 创建章节

点击某个课程，进入课程详情页。

在“添加章节”里输入章节名称，例如：

```text
大模型基础
```

点击“添加章节”。

### 创建课时

在章节下面输入课时名称，例如：

```text
Transformer 原理入门
```

点击“添加课时”。

## 5. 上传和播放视频

进入某个课时页面。

点击“上传视频”，选择本地视频文件。

第一版支持浏览器可以直接播放的视频格式，例如：

- `.mp4`
- `.webm`

上传后视频会保存在服务器本地目录：

```text
uploads/videos
```

网站不会自动转码，所以建议优先上传 `.mp4`。

## 6. 学习进度

在课时页面播放视频时，可以使用：

- “保存进度”：保存当前播放位置
- “标记完成”：把这个课时标记为已完成

下次打开课时页面时，视频会尝试从上次保存的位置继续播放。

## 7. Markdown 笔记

进入课时页面右侧的“Markdown 笔记”区域。

可以直接写笔记，例如：

````markdown
## 今日重点

- Transformer 使用注意力机制
- Prompt 设计会影响模型输出质量

```python
print("hello ai")
```
````

点击“保存笔记”。

右侧/下方的“预览”区域会显示当前笔记内容。

## 8. 时间戳重点

在观看视频时，如果某个时间点很重要：

1. 播放到对应位置
2. 在“当前时间点标题”里输入标题
3. 点击“添加”

之后可以点击这条时间戳笔记，视频会跳回对应时间点。

适合记录：

- 关键概念
- 代码讲解
- 需要反复看的地方
- 老师强调的重点

## 9. 上传附件

在课时页面点击“上传附件”。

可以上传：

- PDF
- PPT
- 图片
- 代码压缩包
- 数据集
- 其他学习资料

附件会保存在：

```text
uploads/attachments
```

上传后可以在课时页面或资料库里查看。

## 10. 资料库

进入：

```text
http://127.0.0.1:3000/library
```

资料库集中展示：

- 所有课时
- 笔记状态
- 附件
- 标签

适合快速查找已经上传或整理过的内容。

## 11. 全文搜索

进入：

```text
http://127.0.0.1:3000/search
```

可以搜索：

- 课程标题
- 课程描述
- 课时标题
- 笔记内容
- 附件标题
- 标签名称

输入关键词后点击“搜索”。

## 12. 作业和项目

进入：

```text
http://127.0.0.1:3000/tasks
```

可以添加作业或项目任务，并绑定到课程或课时。

每个任务可以记录：

- 状态：待完成、进行中、已完成
- 提交链接
- 复盘笔记

适合管理 AI 学习中的实战练习，例如：

- 完成一个 Prompt 案例
- 训练一个小模型
- 阅读一篇论文
- 做一个 RAG Demo

## 13. 错题本

进入：

```text
http://127.0.0.1:3000/mistakes
```

错题本用于集中复习测验中答错的题目。

目前 API 已经预留测验提交和错题记录能力，页面可以查看和标记错题是否掌握。

## 14. 学习统计

进入：

```text
http://127.0.0.1:3000/stats
```

可以查看：

- 总学习时长
- 完成课时数量
- 学习天数
- 最近学习记录

当你保存视频进度或标记完成时，系统会记录学习数据。

## 15. 备份导出

进入：

```text
http://127.0.0.1:3000/settings
```

管理员登录后，点击“下载备份”。

备份内容包括：

- 课程数据
- 章节和课时数据
- 笔记
- 作业任务
- 错题记录
- 上传文件清单

当前第一版导出的是 JSON 备份文件，不会把视频和附件文件本体一起打包。

视频和附件需要额外备份这两个目录：

```text
uploads/videos
uploads/attachments
```

## 16. AI 总结功能

课时页面里有“AI总结/复习题”按钮。

第一版只是占位功能，点击后会提示尚未配置模型 API。

后续可以接入 OpenAI 兼容接口，用于：

- 总结笔记
- 提取重点
- 生成复习题
- 生成行动清单

## 17. 常见问题

### PowerShell 里 npm 报错

如果输入 `npm` 报执行策略错误，请使用：

```powershell
npm.cmd
```

例如：

```powershell
npm.cmd run dev -- --hostname 127.0.0.1 --port 3000
```

### 网站打不开

确认 PowerShell 里已经启动网站，并且显示 `Ready`。

然后访问：

```text
http://127.0.0.1:3000
```

### 上传失败

通常是没有管理员登录。

先进入：

```text
http://127.0.0.1:3000/settings
```

输入管理员密码后再上传。

### 视频不能播放

第一版不做视频转码。请优先上传浏览器支持的 `.mp4` 文件。

如果视频编码特殊，即使后缀是 `.mp4`，浏览器也可能无法播放，需要先用视频工具转换为常见 H.264 MP4。

### 修改 `.env` 后没有生效

修改 `.env` 后，需要停止并重新启动网站。

## 18. 云服务器部署提示

部署到云服务器时，建议：

1. 修改 `.env` 里的 `ADMIN_PASSWORD`
2. 确保 `uploads` 目录有写入权限
3. 定期备份 `prisma/dev.db`
4. 定期备份 `uploads` 目录
5. 使用反向代理和 HTTPS

生产构建命令：

```powershell
npm.cmd run build
```

生产启动命令：

```powershell
npm.cmd run start
```
